Here are the primary commands for project development:

- **Install dependencies:**
  ```bash
  npm install
  ```
- **Run the development server:**
  ```bash
  npm run dev
  ```
- **Create a production build:**
  ```bash
  npm run build
  ```
- **Start the production server:**
  ```bash
  npm run start
  ```
- **Run the linter:**
  ```bash
  npm run lint
  ```
